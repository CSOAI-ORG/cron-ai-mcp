# cron-ai-mcp

MCP server for cron expression tools.

## Tools

- **parse_cron** — Parse and validate cron expressions
- **generate_cron** — Generate cron expressions with presets
- **next_runs** — Calculate next N run times
- **explain_cron** — Explain cron expressions in plain English

## Usage

```bash
pip install mcp
python server.py
```

## Rate Limits

50 calls/day per tool (free tier).
